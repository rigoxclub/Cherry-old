# Cherry

Simple Economy plugin made for Rigox Club network.

## Commands
-  /credits
-  /cherry give
-  /cherry take
-  /cherry reset
-  /cherry set

### Permissions
-  cherry.give
-  cherry.take
-  cherry.reset
-  cherry.set

private plugin of rigox. So you should modify the
code if you want to use it.
